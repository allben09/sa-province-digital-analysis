# 🇿🇦 South African Provincial Digital Economy Analysis

## 🔍 Project Overview
This project analyzes the **digital adoption gap** between South African provinces, comparing Large Enterprises vs. SMEs (SMMEs). 
It highlights how Gauteng leads while Limpopo and Eastern Cape lag behind.

## 📈 Key Visualizations
- **The Gap Chart:** Shows the urban-rural digital divide in SA.
- **The Ranking Chart:** Ranks provinces from "Highly Digital" to "Digitally Lagging".

## 💡 Why This Matters for FNB / SA
South Africa has a massive SMME sector. By identifying which provinces lag, banks like **FNB** can:
- Target rural SMMEs with specific digital banking products.
- Offer training and subsidized internet packages to boost productivity.
- Drive inclusive economic growth.
